# 🎲 Ludo Game in Python

A console-based implementation of the classic **Ludo Game** developed using Python. This project simulates the core gameplay mechanics of Ludo, including dice rolling, player turns, token movement, and winning conditions.

## Features

* Multiplayer gameplay support
* Random dice roll generation
* Turn-based game mechanics
* Token movement according to Ludo rules
* Win condition detection
* Object-Oriented Programming (OOP) implementation
* Simple command-line interface

## Technologies Used

* Python
* Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-link>
   ```

2. Navigate to the project directory:

   ```bash
   cd Ludo-Game
   ```

3. Run the game:

   ```bash
   python main.py
   ```
